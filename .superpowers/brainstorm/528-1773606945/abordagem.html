Como construir o Web App?

Escolha a abordagem que faz mais sentido para você agora

A

Página única simples (HTML + FastAPI)

Um formulário web que coleta briefing do cliente, chama o backend Python e retorna os PNGs em um ZIP para download. Sem banco de dados, sem login. Roda com python app.py.

  • ✅ Pronto em 1 sessão
  • ✅ Zero infra extra
  • ⚠️ Sem histórico de pedidos
B

Chat guiado (estilo WhatsApp)

Interface de chat onde o sistema faz perguntas uma a uma: nicho, cores, logo, restrições, tema. Ao final gera o carrossel. Mais próximo da experiência que você descreveu.

  • ✅ Experiência natural para o cliente
  • ✅ Claude pode pedir esclarecimentos
  • ⚠️ Um pouco mais de código no front
C

Wizard em etapas (passo a passo)

Telas separadas: 1) Identidade visual → 2) Tema/texto → 3) Restrições → 4) Revisão → 5) Download. Barra de progresso visível, upload de logo em etapa dedicada.

  • ✅ Clareza total para o cliente
  • ✅ Fácil de mostrar como produto
  • ⚠️ Mais telas para construir
